# 特么到底算谁的?
<div class="article-head-img">
  <img src="/assets/creation/gg-a-6-0.png"/>
</div>

前些天，有篇文章，收获了一个评论，是建议我用工具检测下文章的AI率。\
我以为（也许是误会）他是想说，那篇文章是AI水的，不由得辩解了下。\
可是回头又一想，特么又不是考试，或是毕业论文什么的，自己博客的议论文啊！\
AI率再高，能高到哪里去？甚至连第二作者都不够格吧？！

但其实另有件事，让我一直很纠结。\
去年夏天，孩子的班主任，让我们家长为她的家访写篇感想。但是交上去后一直没有回复。\
现在想来，是不是和那篇文章一样，也被当成AI文，淘汰了？！

和其它的写作一样，先打了草稿，记述了家访的过程，对比下自己当年，最后抒发感想，做点升华。\
因为孩子的语文老师，唔~~~比较有性格！\
所以我把写完的草稿，分别扔给了几个AI，让它们，“以经验丰富的小学语文老师的角度订正”。\
说来惭愧，什么“的地得”，什么重复描述啊，用词错误啊，语句不通啊... 艾玛！被AI挑出来一堆问题 ... \
咋办？能咋办？！全听它的了！\
一股脑的全采用了。太惭愧了，以至于没注意到，AI甚至还把我孩子的性别，他她的一通乱改！


说回开头的那篇文章，过程也基本一样，先自己起了个大纲。如下↓↓↓

上面那个划掉的，就是。\

（污渍是咖啡~~~！拖延得实在太久了！）\
然后，在编辑器上，一边参考AI的建议，一边敲出来。

再比如，这篇文章的大纲，长成这个样子↓↓↓
<div class="article-head-img">

**所以，自己的主意，加AI辅助的写作，最终的成果，算谁的？** \
**应该，还是算我的吧？！**

要么是我多虑了，那个网友也许只是来推荐/推销小程序罢了。\
于是就在写这篇时，跑去试了下他提到那个小程序工具，啊哈哈！笑西特了！\
好几段竟然有90%，部分甚至95%？！上百字的段落，我只写了不到10个字？！\
最主要的依据是，我引用下：
+ 三段式强迫症
+ 口语化吐槽里突然加入专业的表达
+ 立场中庸

这特么简直就是对我写作风格的精准画像啊！\
（在某个游戏里，我使用了SuperAI的网名，难道说？不能吧？^O^）

最后，我认为AI辅助在创作领域里，文字和2D领域已经非常接近完成时，视频和3D领域还在进行时。
相信到时候，大家的AI率，应该都是我上面的数值吧！
